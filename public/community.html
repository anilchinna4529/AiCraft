How to Contribute

1

Fork the Repo

Start by forking our SDK or documentation repo on GitHub to your personal account.

2

Pick an Issue

Browse open issues labeled good first issue or help wanted.

3

Make Your Changes

Create a feature branch, write clean code, and add tests where applicable.

4

Submit a PR

Open a pull request with a clear description. Our team reviews all PRs within 48 hours.

📋 Contribution Guidelines

  • Follow existing code style and naming conventions
  • Write meaningful commit messages (e.g., feat: add healthcare sector filter)
  • Add tests for new features; don't break existing tests
  • Update documentation if you're changing behavior
  • Be respectful — we follow the Contributor Covenant
